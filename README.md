# TYPE SHIII - Luxury Streetwear Fashion Website

**"Built For Those Who Set Trends, Not Follow Them."**

A world-class, premium luxury streetwear fashion website built with modern web technologies.

## Features

✨ **Stunning Hero Section**
- Animated 3D streetwear character using Three.js
- Cinematic urban environment with neon lighting
- Smooth entrance animations
- Premium campaign aesthetic

🛍️ **Premium Product Display**
- High-quality streetwear collections
- Hover zoom effects
- Quick view and cart integration
- Luxury presentation

🎨 **Modern UI/UX**
- Glassmorphism effects
- Framer Motion animations
- Smooth page transitions
- Mobile responsive design
- SEO optimized

## Tech Stack

- **Frontend**: React 18, Next.js 14
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion, Three.js
- **3D Graphics**: React Three Fiber
- **Icons**: React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit `http://localhost:3000` to see the website.
